# FMCG Distribution Network Optimization

## Overview

This project optimizes product distribution across warehouses and regional demand centers in an FMCG supply chain.

The model minimizes logistics costs while considering:

* Warehouse capacity constraints
* Regional demand
* Transportation costs
* Unmet demand penalties

## Business Problem

FMCG companies often face:

* High transportation costs
* Capacity limitations
* Demand fluctuations
* Distribution inefficiencies

This project demonstrates how optimization techniques can support more efficient supply chain planning.
